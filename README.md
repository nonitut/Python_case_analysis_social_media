# Анализ Posts Dataset. Анализ выполнен с помощью Python, pandas, seaborn и matplotlib.

## Цели исследования

* Определить, какие типы контента вызывают больше всего взаимодействий.
* Проверить, как месяц, день недели и час публикации влияют на охват.
* Сравнить вовлечённость между таргетированными (Paid) и органическими постами.
* Построить матрицу корреляций, чтобы увидеть скрытые зависимости между признаками.

## Основные шаги анализа

1. **Загрузка и предварительный просмотр данных**.
2. **Группировка данных** по типу поста, месяцу, дню недели и часу.
3. **Построение визуализаций** для сравнения средней вовлечённости и охвата.
4. **Корреляционный анализ** между метриками.
5. **Интерпретация графиков**

**Вывод:** Наибольший охват наблюдается в феврале и июле. Сентябрь — самый слабый месяц.

**Вывод:** Вторник и среда показывают лучший охват. Воскресенье — минимум.

**Вывод:** Пики активности в 5, 13 и 14 часов. 19:00 — провал.

## 5. Таргетированные vs органические посты

**Вывод:** Продвигаемые посты получают ощутимо больше взаимодействий.
